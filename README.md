# Series2Image

使用 Python 的 [pyts](https://pyts.readthedocs.io/en/stable/) 库将一维时间序列编码为二维图片，这样可以用于训练卷积神经网络（CNN）

更多详细的内容参考 CSDN 博客：

2. [Python：使用 pyts 把一维时间序列转换成二维图片](https://blog.csdn.net/weixin_39679367/article/details/88653018)
